Mastering the core library for numerical computing in Python

🧠 What is NumPy?

NumPy (Numerical Python) is the foundational package for performing scientific computing with Python. It enables fast mathematical operations on large arrays and matrices and supports advanced operations like broadcasting, reshaping, vectorization, and linear algebra.

Note: NumPy is often the first library used in the data science stack and underpins many others like Pandas, SciPy, scikit-learn, and TensorFlow.

🧭 Mini Project: Grade Analyzer

Use NumPy to build a simple grade analyzer that takes in scores for students and calculates useful statistics.

Step 1: Create a 2D NumPy array of student scores (5 students, 4 subjects)
grades = np.array([[89, 92, 85, 78], [76, 88, 90, 95], [67, 72, 70, 60], [90, 91, 85, 88], [79, 80, 78, 85]])
Step 2: Calculate average score for each student
np.mean(grades, axis=1)
Step 3: Calculate average score for each subject
np.mean(grades, axis=0)
Step 4: Identify the top scorer
np.argmax(np.mean(grades, axis=1))
Tip: Try adding bonus marks and observe how it affects averages using broadcasting!
