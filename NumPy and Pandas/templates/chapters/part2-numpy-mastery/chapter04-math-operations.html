Chapter 4: Mathematical Operations in NumPy

ðŸ§  Concept: Element-wise Arithmetic

NumPy allows you to perform arithmetic operations directly on arrays. These are vectorized and much faster than loops. Operations include +, -, *, /, //, %, and **.

ðŸ§  Concept: Aggregation Functions

Aggregate functions compute summary statistics over arrays: np.sum(), np.mean(), np.std(), np.min(), np.max(), etc. You can specify axes using the axis parameter.

ðŸ§  Concept: Matrix and Dot Operations

Use np.dot() or @ for dot product, and np.matmul() for matrix multiplication. Shape compatibility is crucial in these operations.

ðŸ§  Concept: Trigonometric, Logarithmic, and Exponential

NumPy provides vectorized math functions like np.sin(), np.log(), np.exp(), np.power(), etc. These are optimized and operate element-wise.

# 1. Add two arrays
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
print(a + b)

# 2. Square each element
print(np.power(a, 2))

# 3. Compute mean of a 2D array along axis=0
matrix = np.array([[1, 2], [3, 4]])
print(np.mean(matrix, axis=0))

# 4. Matrix multiplication
A = np.array([[1, 2], [3, 4]])
B = np.array([[2, 0], [1, 2]])
print(np.dot(A, B))

# 5. Trigonometric: Calculate sin of evenly spaced values
angles = np.linspace(0, np.pi, 5)
print(np.sin(angles))

# 6. Use exp and log
data = np.array([1, 10, 100])
print(np.exp(np.log(data)))

# 7. Standard deviation of random array
rand_arr = np.random.randn(1000)
print(np.std(rand_arr))

# 8. Element-wise operations with broadcasting
arr1 = np.array([[1, 2, 3]])
arr2 = np.array([[10], [20], [30]])
print(arr1 + arr2)
