Chapter 5: Linear Algebra with NumPy

ðŸ§  Concept: What is Linear Algebra?

Linear Algebra is the branch of mathematics that deals with vectors, matrices, and linear transformations. NumPy provides the numpy.linalg module to perform advanced linear algebra operations efficiently.

ðŸ§  Concept: Matrix Multiplication

Matrix multiplication is performed using np.dot(), np.matmul(), or the @ operator. Remember, it's not element-wise multiplication.

ðŸ§  Concept: Transpose and Inverse

Transpose is done using .T. Inverse is found with np.linalg.inv(), but it only works for square, non-singular matrices.

ðŸ§  Concept: Determinant & Rank

Use np.linalg.det() to compute the determinant and np.linalg.matrix_rank() for rank. These help understand matrix properties and solvability.

ðŸ§  Concept: Solving Linear Systems

To solve equations like Ax = b, use np.linalg.solve(A, b) where A is the coefficient matrix and b is the result vector.

ðŸ§  Concept: Eigenvalues & Eigenvectors

Use np.linalg.eig() to find eigenvalues and eigenvectors. These are crucial in PCA, physics, and systems modeling.

ðŸ§ª Practice Exercises


# 1. Create 2x2 matrix
A = np.array([[2, 3], [1, 4]])

# 2. Matrix transpose
print(A.T)

# 3. Dot product of vectors
v1 = np.array([1, 2])
v2 = np.array([3, 4])
print(np.dot(v1, v2))

# 4. Matrix multiplication
B = np.array([[5, 1], [2, 2]])
print(np.matmul(A, B))

# 5. Matrix inverse
print(np.linalg.inv(A))

# 6. Check identity property
inv = np.linalg.inv(A)
print(np.round(A @ inv, 2))  # Should be identity

# 7. Determinant
print(np.linalg.det(A))

# 8. Rank of a matrix
print(np.linalg.matrix_rank(A))

# 9. Eigenvalues and eigenvectors
eigenvalues, eigenvectors = np.linalg.eig(A)
print("Eigenvalues:", eigenvalues)
print("Eigenvectors:", eigenvectors)

# 10. Solve system of equations: Ax = b
b = np.array([5, 6])
x = np.linalg.solve(A, b)
print("Solution:", x)
