# 6. Compute row-wise z-score

rows = np.random.randint(1, 100, size=(5, 5))

ðŸ§­ Mini Project: Image Pixel Manipulation (Grayscale Filter)

Goal: Simulate a basic grayscale image filter using NumPy arrays and apply transformations.

Steps:

  1. Create a 100x100x3 array representing an RGB image with random colors.
  2. Convert the RGB image to grayscale using gray = 0.2989*R + 0.5870*G + 0.1140*B.
  3. Normalize grayscale values between 0 and 1.
  4. Apply a threshold: all pixels below 0.5 become 0, above 0.5 become 1.

# Step 1: Create RGB image array
image = np.random.randint(0, 256, (100, 100, 3))

# Step 2: Convert to grayscale
gray = 0.2989 * image[:, :, 0] + 0.5870 * image[:, :, 1] + 0.1140 * image[:, :, 2]

# Step 3: Normalize
gray = gray / 255.0

# Step 4: Apply threshold filter
binary = np.where(gray < 0.5, 0, 1)

print(binary.shape)  # Should be (100, 100)

Bonus: Count total white vs black pixels. Visualize the matrix using a matplotlib heatmap (optional).
