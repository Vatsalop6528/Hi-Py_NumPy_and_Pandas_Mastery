🧪 Code Exercises

# 1. Normalize a dataset
import numpy as np
data = np.random.randint(0, 100, size=50)
normalized = (data - data.min()) / (data.max() - data.min())
print(normalized)
# 2. Detect outliers using Z-score
z_scores = (data - np.mean(data)) / np.std(data)
outliers = data[np.abs(z_scores) > 2]
print("Outliers:", outliers)

🧭 Mini Project: NumPy Weather Analyzer 🌦️

Problem: Build a tool that analyzes weather data for a fictional city over a year and outputs insights.

Step 1: Generate synthetic weather data 
days = 365
temperature = np.random.normal(loc=25, scale=5, size=days)
rainfall = np.random.binomial(1, 0.3, size=days) * np.random.uniform(0, 50, size=days)
Step 2: Identify hottest and coldest weeks 
week_temps = temperature.reshape(-1, 7)
weekly_avg = week_temps.mean(axis=1)
hottest_week = np.argmax(weekly_avg)
coldest_week = np.argmin(weekly_avg)
Step 3: Calculate total rainfall and rainy days 
total_rainfall = rainfall.sum()
rainy_days = np.count_nonzero(rainfall)
Test: Ensure weekly averages match expected size (should be 52 weeks) 
assert len(weekly_avg) == 52
