Series: Think of a Series as a column in a spreadsheet. It has values and an index (labels).

DataFrame: A DataFrame is like a whole spreadsheet â€“ made up of multiple Series aligned by a common index.

import pandas as pd

# 1. Create a Series from a list
s = pd.Series([10, 20, 30])

# 2. Create a Series with custom index
s2 = pd.Series([100, 200, 300], index=['a', 'b', 'c'])

# 3. Access elements by index
print(s2['b'])  # 200

# 4. Create a DataFrame from dictionary
data = {'Name': ['Alice', 'Bob'], 'Age': [25, 30]}
df = pd.DataFrame(data)

# 5. Access column
print(df['Name'])

# 6. Access row by label
print(df.loc[0])

# 7. Add a new column
df['Score'] = [85, 90]

# 8. Filter rows
print(df[df['Age'] > 25])

# 9. Check data types
print(df.dtypes)

# 10. Get DataFrame shape
print(df.shape)

# 11. Get summary stats
print(df.describe())

# 12. Rename columns
df.rename(columns={'Score': 'Marks'}, inplace=True)

# 13. Drop a column
df.drop('Marks', axis=1, inplace=True)

# 14. Set index
df.set_index('Name', inplace=True)

# 15. Reset index
df.reset_index(inplace=True)

ðŸ§­ Mini Project: Student Score Tracker

Goal: Build a small DataFrame that tracks students' names, subjects, and their scores. Then perform some analysis on it.

  1. Create a DataFrame with 5 students and their Math, Science, and English scores.
  2. Add a new column called "Average" that calculates the average score.
  3. Filter students with average score > 80.
  4. Sort the DataFrame by average score.
  5. Display only top 3 scorers with their subjects and average.

import pandas as pd

data = {
    'Name': ['Aarav', 'Diya', 'Karan', 'Meena', 'Ravi'],
    'Math': [78, 88, 92, 85, 67],
    'Science': [80, 90, 85, 88, 70],
    'English': [75, 95, 90, 82, 72]
}

df = pd.DataFrame(data)

# Step 1: Average score
df['Average'] = df[['Math', 'Science', 'English']].mean(axis=1)

# Step 2: Filter students with average > 80
top_df = df[df['Average'] > 80]

# Step 3: Sort by average
top_df_sorted = top_df.sort_values(by='Average', ascending=False)

# Step 4: Top 3 students
print(top_df_sorted.head(3))
