Scenario: You are given a real-world employee dataset and asked 5 rapid-fire questions to test your Pandas skills.

  1. Load employee_data.csv
  2. Find employees with the top 10 salaries
  3. Calculate average salary by department
  4. Identify departments with over 50 employees
  5. Find the percentage of missing values in each column
  6. Create a summary report of all numeric columns

# 1. Load dataset
df = pd.read_csv('employee_data.csv')

# 2. Top 10 salaries
df.nlargest(10, 'salary')

# 3. Average salary by department
df.groupby('department')['salary'].mean()

# 4. Departments with >50 employees
df['department'].value_counts()[df['department'].value_counts() > 50]

# 5. Missing value %
(df.isnull().sum() / len(df)) * 100

# 6. Summary report
df.describe()
