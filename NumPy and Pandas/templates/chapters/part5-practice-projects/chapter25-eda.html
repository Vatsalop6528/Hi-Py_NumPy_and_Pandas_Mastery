ðŸ§  Concept Overview

Exploratory Data Analysis (EDA) is the process of investigating datasets to discover patterns, spot anomalies, test hypotheses, and check assumptions with the help of summary statistics and visualizations. It's the first step in any data science or machine learning workflow.

Core concepts include:

  • Descriptive statistics
  • Missing value handling
  • Distribution and outlier detection
  • Group analysis and correlation
  • Data visualization techniques

Dataset: Use housing_data.csv that includes price, location, bedrooms, bathrooms, size, and age of houses.

  1. Load and preview the dataset
  2. Handle missing values appropriately
  3. Find average house price per location
  4. Detect and remove outliers in price and size
  5. Check correlation between features
  6. Visualize price distribution and price vs. size

# 1. Load data
df = pd.read_csv('housing_data.csv')

# 2. Handle missing values
df.fillna(df.median(), inplace=True)

# 3. Group by location
df.groupby('location')['price'].mean().sort_values(ascending=False)

# 4. Remove outliers in 'price' and 'size'
for col in ['price', 'size']:
    Q1 = df[col].quantile(0.25)
    Q3 = df[col].quantile(0.75)
    IQR = Q3 - Q1
    df = df[(df[col] >= Q1 - 1.5*IQR) & (df[col] <= Q3 + 1.5*IQR)]

# 5. Correlation matrix
print(df.corr())

# 6. Visualizations
import seaborn as sns
sns.histplot(df['price'])
sns.scatterplot(x='size', y='price', data=df)
