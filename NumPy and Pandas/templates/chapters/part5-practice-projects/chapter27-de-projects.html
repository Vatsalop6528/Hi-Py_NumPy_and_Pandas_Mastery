🧭 Mini Project: Build an ETL Pipeline for Online Store

You’re hired as a junior data engineer for an e-commerce platform. Your task is to clean raw files and build a working ETL prototype using Pandas.

  • 🗃️ Load raw users and orders
  • 🧹 Clean both datasets (missing, malformed data)
  • 🔗 Merge on user_id
  • 📊 Calculate KPIs like Total Orders, Avg Value
  • 📤 Save output to cleaned_store_data.csv

💡 Step 1: Load Raw Files

users = pd.read_csv("data/users_raw.csv")
orders = pd.read_csv("data/orders_raw.csv")

💡 Step 2: Clean and Merge

# Basic cleaning
users.dropna(subset=["email"], inplace=True)
orders["amount"] = orders["amount"].fillna(0)

# Merge datasets
combined = pd.merge(users, orders, on="user_id")

💡 Step 3: KPI Calculation + Export

kpis = combined.groupby("user_id").agg({
    "order_id": "count",
    "amount": "mean"
}).rename(columns={"order_id": "total_orders", "amount": "avg_order_value"})

kpis.to_csv("output/store_kpis.csv")
